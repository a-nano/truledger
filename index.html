Trubanc in Lisp

I'm porting Trubanc from PHP to Lisp. This will make it more secure, since it will no longer be necessary to store the bank's private key in plaintext in the file system. It will be stored encrypted, and the bank manager will enter its passphrase when starting the server. It also makes it possible to distribute both client and server as a single executable file on Macintosh, Linux, or Windows. Plus, Lisp is compiled, which will make it a lot faster, though that probably doesn't matter much, since the speed is mostly limited by the file system and the crypto.

Git repository at github.com/billstclair/trubanc-lisp

To download the code to the "trubanc-lisp" subdir of pwd:

git clone git://github.com/billstclair/trubanc-lisp.git trubanc-lisp
