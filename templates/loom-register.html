This page creates an account in this client to use as a client for one of more Loom servers/wallets. Enter your desired passphrase in both the "Passphrase" and "Verification" fields. Enter a name for your account in the "Acount Name" field. Enter a Loom server URL as the "Loom Server". If this is a new wallet on that server, paste an "Invitation". If you want your wallet to be accessible via the Loom web server, leave the "Private" box unchecked. If you check "Private", your Loom wallet will only be accessible from the Truledger web client, and the Loom server will see the wallet information only as encrypted nonsense. Finally, click the "Create Account" button to create a new account. The "Login" button allows you to log in to an existing account on this client web server with your "Passphrase".
